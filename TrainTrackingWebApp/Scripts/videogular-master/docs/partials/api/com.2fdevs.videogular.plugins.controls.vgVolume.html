vgVolume
directive in module com.2fdevs.videogular.plugins

Description

Directive to control the volume. This directive acts as a container and you will need other directives like vg-mutebutton and vg-volumebar to control the volume. In mobile will be hided since volume API is disabled for mobile devices.

<videogular vg-theme="config.theme.url">
   <vg-video vg-src="sources"></vg-video>

   <vg-controls vg-autohide='config.autohide' vg-autohide-time='config.autohideTime'>
       <vg-volume></vg-volume>
   </vg-controls>
</videogular>

Usage

as element:
<vg-volume>
</vg-volume>
