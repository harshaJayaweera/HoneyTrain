vgDash
directive in module com.2fdevs.videogular.plugins

Description

Adds support for vg-video and vg-audio tags. This plugin requires dash.all.js file available at dash.js project: https://github.com/Dash-Industry-Forum/dash.js

<videogular vg-theme="config.theme.url" vg-autoplay="config.autoPlay">
   <vg-video vg-src="sources" vg-dash></vg-video>
</videogular>

Usage

as element:
<vg-dash>
</vg-dash>
